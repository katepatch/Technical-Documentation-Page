Introduction

HTML and CSS are great ways to create a single page web application.

Please refer to this documentation with any questions you may have regarding HTML and CSS.

HTML

HTML is the short way of saying Hypertext Markup Language. This is the foundation of websites. HTML uses tags like:

  • headings
    header
  • paragraphs
    p
  • images
    img
  • and more!
    section
    div
    span
to define different elements on the webpage. HTML can be used in conjuntion with other languages like CSS to create dynamic and interactive web pages.

CSS

Cascading Style Sheets is a language used to present the layout of HTML. It is an essential tool for web designers to create user friendly websites.

With CSS you can define the visual appearance of web pages by including:

  • Fonts
  • Colors
  • Layouts
to make it visually appealing.

Responsive Design

There are many ways for a user to access a web design. Be it a desktop, tablet, or mobile phone these devices vary in screen sizes.

With responisve design you can ensure that your website's laytout can respond to the size of the screen to make it user friendly, readable, and adjustable.

Using CSS you can create a responisive design by adjusting all elements within your web page as it shifts screen sizes

Application

By using HTML, CSS, and even JavaScript you can create a simple web application.

Web applications have a wide range of uses. From simple games and single page sites to more complex like an online banking system.
